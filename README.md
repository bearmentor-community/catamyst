## Tech Stack

Available in more details at [a.catamyst.com/stack](https://a.catamyst.com/stack) or [catamyst.com/stack](https://catamyst.com/stack) (Coming Soon).

### All

- **TypeScript**, including:
  - TSX
  - JSX
  - JavaScript
  - Node.js
  - HTML
  - CSS
  - [The Web Platform](https://mdn.dev)
- **Prettier** code formatter
- **ESLint** code linter
- **GitHub** code repository hosting
- **Linear** issue tracking and management

### Web

- **React**
  - UI library/framework
- **Remix**
  - **ESBuild** as bundler
  - **Babel** as transpiler
- Styling packages:
  - **Stitches** for CSS-in-JS and themes
  - **Radix UI** for accessible component
- Data fetching options:
  - Remix built-in loader and action with **fetch** or **Axios**
  - **React Query**
  - **urql**
- Hosting options:
  - **Cloudflare Pages**
  - **Vercel**

### Android

- **React Native**
- **Kotlin**

### iOS

- **React Native**
- **Swift**

### API

- **Express**
- **GraphQL**
- **Prisma**
- **Nexus GraphQL**
- Database options:
  - **PostgreSQL** hosted in **Supabase**
  - **MySQL** hosted in **PlanetScale**

### Test

- Unit and Functional:
  - **Jest**
  - **React Testing Library**
  - **Tomato**
- End to End:
  - **Cypress**

### Other

- Domain:
  - **Cloudflare Registrar**
- DNS:
  - **Cloudflare DNS**
- Short link:
  - **Rebrandly**
- Analytics options:
  - **Splitbee**
  - **Fathom**
  - **Simple Analytics**
- Error tracking options:
  - **Sentry**
  - **Replay**
- Marketing email options:
  - **Buttondown.email**
  - **Revue**
- Monitoring options:
  - **Instatus**
  - **UptimeRobot**
- Transactional email options:
  - **Sendgrid**
  - **Amazon SES**
  - **Postmark**
- Containerization:
  - **Docker**
